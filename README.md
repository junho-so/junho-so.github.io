# Junho So Homepage

Static personal homepage for `junho-so.github.io`.

The layout is designed as a lightweight Jon Barron-style academic homepage:
profile, research summary, publications, education, awards, honors, and skills.
Publication thumbnails are cropped from the corresponding paper PDFs.

## Deploy on GitHub Pages

1. Create the repository `junho-so.github.io` under the `junho-so` account.
2. Upload these files to the repository root.
3. In the repository settings, open **Pages** and publish from the `main` branch root.

The site will be available at `https://junho-so.github.io`.
